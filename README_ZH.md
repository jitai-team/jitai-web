## ✨ 核心特性

- **门户系统**: 标准门户、空白门户，支持导航菜单配置
- **页面管理**: 常规页面、AI数据管理页面、AI数据分析页面、数据录入页面
- **丰富组件**: 表格、表单、模型筛选器、统计图、弹窗、级联表、级联树、画廊、看板、列表、时间轴、树形、日历
- **自定义控件**: 支持自定义 React 控件扩展
- **全局样式**: 统一的样式管理系统
